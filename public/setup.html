Remote Trigger API

Your room PC sends a plain HTTP POST to start or stop recording. No authentication required — keep RecordHub on your local network.
POST http://<recordhub-ip>:3299/api/trigger
{
  "action": "start",
  "title": "Talk Title",
  "presenter": "Jane Smith",
  "room": "BallroomA"
}
POST http://<recordhub-ip>:3299/api/trigger
{ "action": "stop" }
{ "ok": true, "session": { … } }
Returns ok: false with an error string on failure.
