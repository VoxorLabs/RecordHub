Remote Trigger API

Any device on the LAN can start or stop recording with a plain HTTP POST.
POST http://<recordhub-ip>:3299/api/trigger
{
  "action": "start",
  "title": "Talk Title",
  "presenter": "Jane Smith",
  "room": "BallroomA"
}
POST http://<recordhub-ip>:3299/api/trigger
{ "action": "stop" }
