Trigger API

RoomAgent uses these endpoints to start and stop recording. Any device on the LAN can call them.

POST http://<this-machine-ip>:3299/api/trigger { "action": "start", "title": "Talk Title", "presenter": "Jane Smith", "sessionId": "123" }
POST http://<this-machine-ip>:3299/api/trigger { "action": "stop" }
